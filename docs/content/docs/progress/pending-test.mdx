# 待测试

- 火山方舟 Seedance 预置模型：Volcengine Provider 内置模型列表已更新为 Seedance 2.0、Seedance 2.0 Fast、Seedance 1.5 Pro、Seedance 1.5 Lite、Seedance 1.0 Pro，并保留自定义模型输入；Ark Base URL 配置档可从下拉选择预置模型，AgentPlan Base URL 配置档仍保持手动输入模型 ID。
- 画布页设置入口：画布详情页顶部用户操作区现在可直接打开“设置”；底部工具栏原“画布外观”按钮改为“设置”，并将主题模式、网格样式和图片信息开关迁移到设置弹窗的“常规”分区。
- 图像节点模型选择：图像节点 prompt 面板已改为参考图 2 的卡片式输入面板，上方保留“标记”和常驻“参考”入口；点击“参考”进入选择状态，画布顶部显示“从画布选择参考”提示条，可点击“定位节点”返回发起选择的原图像节点，可点击“退出”或按 `Esc` 结束选择；选择状态下点击画布空白不会退出，可继续选择参考图；悬浮已有图片节点时居中显示“选择为参考”，点击未选图片节点会创建到当前图像节点的数据连线并显示为参考缩略图，选择态会保持，可继续点击其他图片节点追加参考；已选参考节点会叠加淡化蒙版，悬浮时居中显示红色“取消参考”，点击后移除该参考连线；缩略图右上角 `x` 也可删除对应数据连线；中间为大输入区，底部保留提示词库、供应商/Profile + 模型两列选择、图像参数、点数和生成按钮；新建图像节点会带入当前图片默认模型快照；节点保留 `providerOverride` 快照并在生成时优先使用；节点/默认 Profile 可直接驱动图像生成，不再被旧配置 API Key 前置检查拦截；Profile 被禁用或删除时节点显示警告、面板提示不可用并禁用生成按钮。
- 图像节点生成张数：生成张数已从图像设置面板拆成底部独立小下拉，参考图 2 的紧凑样式，只显示 `1张` 到 `4张`，图像设置面板内不再显示生成张数区域。

- 画布节点连线交互：节点左右两侧连接点改为圆形 `+` 热区，无需先悬浮到节点面板，鼠标贴近节点外侧 `+` 附近即可显示并拉出数据线；`+` 会在小范围内跟随鼠标并在离开后复位；拖出连线时隐藏节点两侧连接点，并将活动线条移动收敛为帧级轻量更新，减少拖线过程中的卡顿。
- 节点级模型选择基础设施：画布节点新增 `providerOverride` 模型快照；新建图像、视频、音频、文本节点会复制当前全局默认模型；Provider 路由按节点 override、全局 default、legacy 解析，禁用/删除 Profile 会给出明确错误；新增独立 `NodeModelSelector` 组件，当前已先接入图像节点。
- Provider Profile 模型同步：每个配置档卡片删除按钮左侧新增“同步远端模型”按钮，点击后通过统一 ProviderClient 拉取远端模型；没有远端列表接口的 Provider 回退 manifest 模型，并把模型 ID 按严格大小写去重合并到当前 Profile 的 `models` 字段，保留旧顺序、新模型追加，关闭设置弹窗会取消进行中的同步。
- Provider 模型列表 SDK：ProviderClient 新增 `listModels(providerId, profileId?)`，OpenAI Compatible 可通过 Profile 认证调用 `/v1/models` 拉取远程模型；未实现远程列表的 Provider 会回退到 manifest 声明模型。
- Provider Profile 模型缓存：支持远程模型列表的配置档保存后会异步拉取并缓存模型列表，配置档卡片展示未加载、已加载数量/更新时间或失败状态，并提供手动刷新入口；刷新失败时保留旧缓存。
- Provider 管理设置面板：顶部导航和移动端导航新增“设置”入口；设置弹窗内提供“AI 服务商”和“默认模型”section，可新增、编辑、删除、启停 Provider Profile；默认模型页面以文本、图片、视频、音频四行表格配置默认 Profile 和模型，模型选择优先使用 Profile 缓存下拉，其次使用 Provider 内置模型下拉，最后回退手动输入，并支持自动保存和首次启用提示。
- Provider 任务生命周期：生成任务会记录到本地 TaskStore，支持刷新后识别待写回/不可恢复任务；删除节点会中断关联请求并清理任务；参考图无法跨刷新恢复时会给出降级提示。
- Codex App 插件文档：新增插件安装教程，区分 AI 自动安装、手动安装和本仓库开发调试安装；同步优化插件 README 与技能说明。
- 配置与用户偏好：每个模型渠道新增 OpenAI / Gemini 调用格式选择，默认 OpenAI，切换默认格式时同步更新默认 Base URL；渠道 Tab 增加紧凑醒目的提醒，说明模型是否可选需要到“模型”Tab 选择。
- Gemini 调用格式：支持拉取 Gemini 模型列表，并用于文本问答、在线 Agent 工具调用、基础生图和图生图请求。
- 音频和视频生成：选择 Gemini 调用格式时先给出不支持提示，仍需使用 OpenAI 兼容渠道。
